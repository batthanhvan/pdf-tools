# PDF Tools

Batch scripts for PDF processing using Ghostscript library. Works well
even with very big and heavy PDF files. No file size limits.

---

## PDF to JPG Converter

Convert PDF to JPEG images using the Ghostscript library.

**How to use:**   
Drag and drop your PDF file onto _pdf-to-jpg.bat_

![how-to-use](https://user-images.githubusercontent.com/4609437/151794255-96fe05bf-1b61-4e1e-b168-e8742933c308.gif)

---

## PDF Combiner

Combine multiple PDF files into a single PDF. Perfect for merging tables, figures, or manuscript PDFs for journal submissions.

**How to use:**   
Drag and drop multiple PDF files onto _combine-pdf.bat_

**Output:**
- Output filename is `combined.pdf`
- If the file already exists, a numeric suffix is added: `combined_1.pdf`, `combined_2.pdf`, etc.
